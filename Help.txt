Command: <b>Keybinding</b>
    <i>Description</i>
------------------------------------------------------------------------------------
Start/Stop: <b>Space</b>
    <i>Launches or pauses the simulation.</i>

Step: <b>1</b>
    <i>Performs one single update.</i>

Clear: <b>2</b>
    <i>Clears the board and prepares a new simulation for being run.</i>

Random: <b>3</b>
    <i>Randomly fills half the surface of the board with alive cells.</i>

Reset: <b>4</b>
    <i>When a file is open and modified, discards any change.</i>

Glider gun: <b>5</b>
    <i>Draws a gun that eternally launches gliders.</i>

Spacefiller: <b>6</b>
    <i>Draws a special figure that fills the board.</i>   

Open: <b>Ctrl+O</b>
    <i>Opens a file and initalizes the simulation with its content.</i>

Save as: <b>Ctrl+S</b>
    <i>Saves the current state of the simulation in a text file,   
    which can then be opened to resume/launch a simulation.</i>

Quit: <b>Ctrl+Q</b>
    <i>Quits the application.</i>

Zoom in: <b>Ctrl++</b>
Zoom out: <b>Ctrl+-</b>
Reset zoom: <b>Ctrl+0</b>
    <i>Resets the zoom level to 0 (100%).</i>

Show Grid:
    <i>Displays a light gray grid over the board.  
    It can help drawing cells on the board with the mouse.</i>

Dark Theme:
    <i>Switches to dark theme when ticked, to light theme when unticked.</i>

Stability Detection: <b>Ctrl+T</b>
    <i>Enables/Disables the S.D. (Stability Detection) mode. In this mode,  
    the simulation stops as soon as the evolution stabilizes over time.</i>

Pan left: <b>H</b>
Pan down: <b>J</b>
Pan up: <b>K</b>
Pan right: <b>L</b>
  
World size  
=> Extend: <b>Ctrl+PageUp</b>
    <i>Increases the size of the board (virtual world) by 50px.</i>
=> Shrink: <b>Ctrl+PageDown</b>
    <i>Decreases the size of the board by 50px.</i>
  
Refresh:
    <i>Allows you to change the speed of the simulation.  
    The smaller the rate, the higher the fps.   
    Be sure to stop and restart the simulation for the change to take effect.</i>
