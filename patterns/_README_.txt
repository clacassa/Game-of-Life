== CONWAYLIFE.COM PATTERN DATABASE ==
Database compiled on: Wed, Sep 27 2023
Downloaded from: conwaylife.com/patterns/all.zip

This pattern database contains 7577 pattern files for several hundred patterns in Conway's Game of Life and related cellular automata. All patterns are provided in the RLE file format.

RLE (.rle): A short way on encoding patterns, but not very human-readable. See conwaylife.com/wiki/RLE
